盒模型

盒模型就是一种网页中的思维模式，可以把每一个元素都当成一个盒子，这个盒子，我们可以用手机盒子来举例，手机就是我们盒模型中的“content”，白色的填充泡沫就是我们盒模型中的“padding”，手机盒子的厚度，就是我们盒模型中的“border”，盒子与盒子之间的距离，就是我们盒模型中的“margin”。

一个盒模型包括：content、padding、border、margin

计算总盒子的height：content的height+2*padding（上下padding）+2*border（上下border）

计算总盒子的width：content的width+2*padding（左右padding）+2*border （左右border）

盒模型主要包括哪两种模式

  • ie盒模型：box-sizing：border-box；我们代码中设置的width（宽），height（高），就是总盒子的大小，然后里面的content部分的宽和高，浏览器会自动给我们计算。

  • 标准盒模型：box-sizing:content-box;(默认的)，我们代码中所写的width和height指的只是content部分的宽度和高度，盒子的总大小就按照上面的计算公式计算。

       .box{

            width:200px;

            height:200px;

            background:green;

            padding:20px;

            border:10px solid gold;

            margin-bottom:20px;

            box-sizing: border-box;

        }

margin 值穿透

一个父盒子，里面有一个子盒子，如果父盒子没有设置border-top或者padding-top的时候，当我们给子盒子设置margin-top的时候，父盒子会跟着子盒子一起向下移动。

解决方法：
1、给父盒子添加一个上边框，border-top

<style>

        *{

            margin:0;padding:0;

        }

        .box{

            width:200px;

            height:200px;

            background:green;

            border-top:1px solid transparent; // transparent 透明颜色

        }

        .child{

            width:100px;

            height:100px;

            background:pink;

            margin-top:20px;

        }

    </style>

2、既然margin-top存在兼容问题，那咱们也可以不用，换成padding即可。
为什么要加box-singzing:border-box;当我们给父盒子添加padding的时候，盒子的总大小，就会改变，为了让总盒子的大小还是我们代码中设置的大小，所以加的。
常用

            margin:0;padding:0;

        }

        .box{

            width:200px;

margin 值垂直方向合并

有两个同级的元素，我们给第一个盒子设置margin-bottom的时候，比如margin-bottom：30px; 给下面的盒子设置一个margin-top：40px; 按理说应该两个盒子总共的距离是70px;但是实际情况不是这样的，会出现在垂直方向上的margin值合并。会取最大的值。最终两个盒子的距离是40px;

<style>

        *{

            margin:0;padding:0;

        }

        .box1,.box2{

            width:200px;

            height:200px;

            background:green;

        }

        .box1{

            margin-bottom:30px;

        }

        .box2{

            margin-top:40px;

        }

    </style>

解决办法：避免这种情况就行了，单独写一个margin或者你给他们设置同方向的就行了。
