第七天css笔记

background

1 background-color: 背景颜色
2 background-image:背景图片
3、background-position:背景图片的位置
4、background-attachment:scroll（背景图片相对于元素是固定的） 或者fixed（背景图片相对于窗口是固定的）
5、background-size：百分比/具体的像素值/cover/contain

 <style>

        .box{

            width:600px;

            height:600px;

            border:1px solid red;

            /*背景颜色*/

            background-color:pink;

            /*背景图片*/

            background-image:url("images/apple.jpg");

            background-repeat: no-repeat;

            /*

            1、

              水平位置：left center right

              垂直位置：top center bottom

            2、具体的像素值



            3、百分比

              */

            /*background-position:center center;*/

            /*background-position: 30px 30px;*/

            background-position:50% 50%;

        }

    </style>

cover 这个值也是放大图片，有可能放大到一定的程度，图片的内容会超出盒子的大小，就不能显示完全
contain：缩放到最合适的比例，只要有一边触碰到盒子的大小，就会停止缩放

 <style>

        .box{

            width:400px;

            height:300px;

            border:1px solid red;

            background-image: url("images/head.gif");

            background-repeat: no-repeat;

            background-position:center center;

            /*background-size:cover;*/

            background-size:contain;

            /*background-size:300px 300px;*/

            /*background-size:100% 100%;*/

            /*复合属性，注意：background-sie 不能复合到里面写，要单独写*/

            background:pink url("images/apple.jpg") center center  no-repeat ;

        }

    </style>
