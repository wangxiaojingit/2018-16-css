##第九天css笔记

定位：

  • 默认的是static

  • 相对定位

  • 绝对定位

  • 固定定位

相对定位：relative

特点：（方向值：left、right、top、bottom）

  • 没有脱离文档流

  • 参照物是自身

  • 相对定位的元素，它的层级大于普通文档流元素的层级（z-index）

  • 当给相对定位的元素同时设置四个方向的值，lef、top、right、bottom的时候，它是按照left、top这两个值进行的。

  • 最重要，最常用的一个作用，就是给绝对定位的元素做参照物。

 .box{

            width:100px;

            height:100px;

            background:green;

            /*相对定位*/

            position:relative;

            top:100px;

            left:100px;



        }

绝对定位：absolute

绝对定位的使用：想让谁绝对定位，就给谁添加position:absolute;同时设置一下方向值，让谁做它的参照物，就给谁添加position:relative;
绝对定位的特点：

  • 1、脱离文档流

  • 2、设置绝对定位元素的层级大于普通文档流的层级

  • 3、如果你不设置参照物的时候，它的参照物是body，如果设置了参照物，它的参照物有一个标准，必须是上级元素，不能是同级元素

  • 4、 查找参照物的时候，只要元素中有position：relative，或者是position:absolute,或者是position：fixed，三者中的任何一个，都可以作为参照物的标准。 也就是说，只要有position：relative、position:absolute、position：fixed中的任何一个就已经满足参照物的特点了，也就不用再设置position：relative。

  • 5、浏览器在查找参照物的时候，是按照就近原则，如果说，绝对定位的元素，它的父亲元素拥有参照物标准（上面的三者之一），它的爷爷元素也拥有，但是浏览器是按照“就近原则”，也就是离它最近的那个就是它的参照物。

  • 6、绝对定位的元素，如果不设置宽度，它的宽度就是自身内容的宽度。如果绝对定位的元素，它的宽度用百分比的时候，它的宽度是参照物，不是父级。

固定定位

position:fixed;
固定定位主要用来做快速导航，想要谁固定定位，就给谁加position:fixed;它也有四个方向值，它的参照不用设置，就是浏览器窗口。始终相对浏览器窗口不变的。

面试题：

如何让一个元素在一个盒子中或者是屏幕中水平居中：

给一个元素设置绝对定位，然后设置left：50%；margin-left：负的盒子宽度的一半。

       *{

            margin:0;padding:0

        }

        .box{

            width:100px;

            height:100px;

            background:pink;

            position:absolute;

            left:50%;

            top:0;

            margin-left:-50px;

        }
如何让一个元素在一个盒子中或者是屏幕中垂直居中：

让一个元素绝对定位，然后设置top：50%；margin-top:负的盒子高度的一半

     .box{

            width:100px;

            height:100px;

            background:pink;

            position:absolute;

            left:0;

            top:50%;



            margin-top:-50px;

        }
如何让一个元素在一个屏幕中水平垂直居中：

left:50%;top:50%; margin-left:负的盒子的宽度的一半，margin-top：负的盒子高度的一半

 .box{

            width:100px;

            height:100px;

            background:pink;

            position:absolute;

            left:50%;

            top:50%;

            margin-left:-50px;

            margin-top:-50px;

        }
