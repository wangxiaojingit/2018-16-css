第二天css笔记

1、网页的基本组成

  • html: 超文本标记语言，它不仅可以单纯的表示文本类，还可以表示音频、视频、图片等。（骨架）

  • css：层叠样式表，美化html的（衣服，化妆品）

  • js：轻量级的编程语言，（行为和动态）

2、html的基本结构

<!DOCTYPE html> //文档声明

<html>      //html 是根元素

  <head>    

      <meta charset="UTF-8"> // 全国通用的编码，不写会乱码

      <meta name="keywords" content="关键词"> //放关键词的，给搜索引擎看的

      <meta name="description" content="描述的内容"> // 网页描述，给搜索引擎看的

      <title>标题</title> // 网页标题

  </head>

  <body>



  </body> 



</html>

3、html标签的特点

  • 关键词用尖括号包括起来

  • 有开始标签和结束标签组成，成对出现

  • 结束标签比开始标签多了一个反斜杠

大部分标签都符合上面的特点，还有一类特殊的标签，被叫做单标签，也被称作“自闭合标签”，比如说

     <img  />

清除元素自带的margin和padding

   <style type="text/css">

        /*

          *通配符选择器，代表选中所有的标签，margin值和padding值设置为0

        */

        *{

            margin:0;

            padding:0;

        }

       div{

           width:100px;

           height: 100px;

           background:gold;

       }

    </style>

清除ul，ol列表的默认样式

      /*

        清除ul、ol前面的小圆点或者数字

       */

        ul,ol{

            list-style:none;

        }




  <style type="text/css">

        /*清除a标签的下划线*/

        a{

          text-decoration:none ;

        }

    </style>
