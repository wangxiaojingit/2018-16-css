3、让一个块级元素水平居中的方法：

<style type="text/css">

       /*标签选择器

         margin:0 auto; 代表的意思：上下的margin值为0，左右自动计算，写完这个属性，这个盒子就会在父元素的盒子中水平居中，这个是块级元素经常用到的。



       */

        div{

           margin:0 auto;

        }

    </style>

5、通常一些标签会带有默认的margin值或者padding值，会有白色的多余的块，咱们在写页面的时候，通常不需要，所以我们需要先把他们清除掉，在后面按照需求，咱们自己再设置。

  • 被叫做“通配符”，代表所有的元素，

<style type="text/css">

       /*

        * 代表通配符，选中所有的元素



       */

        *{

            margin:0;

            padding:0;

        }

        p{

            background-color:green;

            text-align:center;

        }

    </style>

6、文字缩进

            text-align:center;

            text-indent:2em;

        }

7、清楚列表中默认带的样式，不管是ul，还是ol，我们在做项目的时候，通通不用它的默认样式，因为没有办法对它进行控制，我们会自己去按照设计需求去做。如何清除？

    <style type="text/css">

        ul,li,ol{

            list-style:none;

        }

    </style>

8、清楚a标签默认自带的下划线

 <style type="text/css">

        a{

        }

    </style>

9、什么是块级元素，什么行内元素，什么是行内块？

块级元素的特点：
1、很霸道，独占一行，不允许其他元素和他共占一行;
2、可以设置width，height，能够起作用；
3、排列方式：从上到下

行内元素的特点：
1、可以和其它的行内元素共占一行
2、不修改dispaly的情况下，就是说默认情况下，给它设置宽、高，不起作用
3、从左到右

行内块的特点：例如：img
1、可以和其它的行内元素共占一行
2、给它设置宽、高，起作用
3、从左到右

11、 display，块级元素和行内块元素的相互转换：

  • 行内元素转换块级元素：display：block;

  • 行内元素转换行内块： display：inline-block;

  • 转换为行内元素： display：inline；//这个基本不用，在做项目的时候如果是行内元素，直接用行内标签即可

  • 消失 ：display：none

3、面试题：让一个元素消失的几种方法

1、display：none; // 常用
2、 visibility: hidden; // 常用
3、改变透明度：opacity: 0; // 常用
4、改变背景色（只是取巧，除非是没文字内容的);

display:none 和 visibility: hidden 有什么区别？
display:none, 不占位，visibility: hidden 虽然消失不见了，但是它的位置还在，一个控制dom元素，一个控制css
