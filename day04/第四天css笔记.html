选择器

1.标签选择器

权重：1
标签选择器，就是咱们之前说的标签关键词，写什么标签，就会把哪一类标签选中
下面代码的意思就是把所有的p段落选中

 <style>

        p{color:gold;}

 </style>





<body>

   <p>段落p</p>

   <p>段落p</p>

   <p>段落p</p>

   <p>段落p</p>

   <p>段落p</p>

   <h2>标题</h2>

</body>

2、通配符选择器

权重：1
通配符选择器，用“*” 来表示，它代表选中所有的元素。

    <style>

        *{margin:0;padding:0;}

    </style>

3、属性选择器

权重：10
用法：1把特定属性用 ，具有这一类的属性标签都会起作用
用法：2 把特定属性=特定属性值 用，具有这一类的属性等于属性值的标签都会起作用
例如：

   <style>

        [title]{

              width:100px;

          }

        p{

            background:green;

        }

        [title="p1"]{

           color:red;

        }

    </style>

  <p title="">段落p</p>

  <p title="p1">段落p</p>

4.类选择器

权重：10
你要选中谁，就用class给那个标签起名字
使用的时候，用“.”+你起的名字

   <style>

        .box{

            width:100px;

            height:100px;

            background:green;

        }

    </style>



<body>

  <div class="box"></div>

</body>

5、id选择器

权重:100
你想选中谁，就给谁身上通过id给它起一个名字
用id选择器的时候，它就相当于我们的身份证，每个人只有一个特定的号，其它人的身份证号不能给我们的相同，意思就是说，假如我用id=“box”，一个页面中用id起的名字叫box的只能有一个。其它元素还可以用id，但是不能叫box这个名字。
使用：用“#”+你起的名字

   <style>

        #box{

            width:100px;

            height:100px;

            background:green;

        }

    </style>



   <div id="box"></div>

   <div id="box1"></div>

   <div id="box2"></div>

   <div id="box3"></div>

6.子选择器

权重：所有选择器之和
通俗的说就是“儿子选择器”，我们在使用的时候，会把父亲元素和子元素通过“>”连接起来，代表，把父元素下面的这一类的儿子元素都选中
例如：

<style>

        ul>li{

           background:pink;

        }

        .main>p{

            color:red;

        }

    </style>

8、分组选择器

权重：所有选择器之和
如果具有相同样式的标签，我们为了减少代码量，也为了优化，可以通过分组选择器来做，把选中的元素，用“，”（逗号）隔开，就代表选中了所有的。
例如：

    <style>

        p,.box{

            color:green;font-size:14px;

        }

        /*p{color:green;font-size:14px;}*/

        /*.box{color:green;font-size:14px;}*/

    </style>

9、相邻兄弟选择器

权重：所有选择器之和
把紧邻着它的特定兄弟元素选出来
连接符：+
例如：

    <style>

        h1+h3{

            color:red;

10、通用兄弟选择器

权重：所有选择器之和
不管有没有相邻，就会把符合条件的所有的兄弟元素选中
连接符“~”
例如：

    <style>

        h1~h3{

            color:red;

        }

    </style>

table (表格)

table 代表格，每一行用tr来表示，每一个单元格用td来表示，如果是表格的表头，td换成th，这样文字会加粗（其实写成td也没错，只是更加语意话，更好）；表格的名字，用caption

完整的table，有thead、tbody、tfoot



<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>table 表格</title>

</head>

<body>

   <table>

       <thead>

            <caption>成绩表</caption>

          <tr>

              <th>语文</th>

              <th>数学</th>

              <th>外语</th>

          </tr>

       </thead>

       <tbody>

          <tr>

              <td>100</td>

              <td>100</td>

              <td>100</td>

          </tr>

          <tr>

              <td>100</td>

              <td>100</td>

              <td>100</td>

          </tr>

       </tbody>

       <tfoot>

         <tr>

             <td>总计：</td>

             <td>总计：</td>

             <td>总计：</td>

         </tr>

       </tfoot>

   </table>

</body>

</html>

table几个重要的属性

  • border 边框

  • cellpadding:单元格的内填充

  • cellspacing:单元格与单元格之间的间距

  • rowspan：合并行

  • colspan：合并列

table几个重要的样式

  • 合并边框线：border-collapse：collapse

  • 平均分布：table-layout:fixed;(写这个的前提，必须写width)

面试题：完整的table写法和简写的区别？

完整的写法，已经清清楚楚的告诉了浏览器，哪个是表格的头部，哪个是表格的身体，哪个是表格的尾部，即使我们把它们的代码顺序颠倒，在浏览器上依然显示正确的顺序。
除此之后，更加的语义化，更利于搜索引擎去搜索。更符合seo优化。
