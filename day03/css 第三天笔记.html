css 第三天笔记

给元素设置宽度 高度

   <style>

        span{

            width:100px; //宽度

            height:100px; //高度

            background:green;

        }

    </style>

文字大小：font-size

   div{

     font-size:16px;

   }

文字颜色：color

    p{

      color:red;

    }

文字水平居中

left(左对齐)、center（居中对齐）、right（右对齐）

    p{

        text-align:center;

    }

文字垂直居中

height（高度）等于line-hegiht（行高）



    P{

       height:100px;

       line-height:100px;

    }

给盒子设置边框

<style>

        div{

            width:100px;

            height:100px;

            /*

              第一个值代表的是边框的宽度

              第二个值代表的是边框的样式，solid 代表实线

              第三个值代表的是颜色

            */

            /*border:1px solid red;*/

            border-bottom:1px solid red;/*下边框 */

            border-top:1px solid red; /* 上边框*/

            border-left:1px solid red;/*左边框*/

            border-right:1px solid red; /*右边框*/

        }

    </style>

清除网页中各个元素自带的margin和padding

*{margin:0;padding:0}

清除a标签自带的下划线

   a{

      text-decoration:none;

   }

清除ul，ol列表前面默认的样式

   ul,ol{

      list-style:none;

   }

让一个盒子在它的父级中水平居中

    div{

       margin:0 auto;

    }

文字缩进：text-indent：值

 <style>

        /*

          文字缩进：text-indent：值

        */

        p{

            height:50px;

            line-height:50px;

            background:green;

            text-indent:20px;



        }

    </style>

块级元素

  • 1、 块级元素，很霸道，独占一行，不让其它的元素和它在一行。

  • 2、排列方式：上下排列

  • 3、设置宽高的时候，可以起作用

行内元素

  • 1、行内元素允许其它的行内元素和它共占一行

  • 2、排列方式从左到右

  • 3、默认情况下，设置宽高不起作用,它的宽高是里面的文字内容的大小

行内块元素

  • 1、行内元素允许其它的行内元素和它共占一行

  • 2、排列方式从左到右

  • 3、默认情况下，设置宽高起作用

标签汇总：

  • 标题：h1-h6

  • 段落：p

  • 超链接：a

  • 大盒子：div

  • 小盒子：span

  • 图片：img

  • 粗体：b 或者strong

  • 斜体：i或者em

  • 删除线：del

  • 三大列表：ol ul dl（li，dt，dd）

    块级元素：h1-h6、p、div
    行内元素：a、span、strong、b、i、em、del、big、small
    行内块：img

    display的值

  • display:block; 转换成块级元素

  • display：inline-block；转换成行内块元素

  • display：inline；转换成行内，但是这个咱们通常不用，如果要转换，直接用行内元素即可

  • display：none；让一个元素消失。

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>display</title>

    <style>

        span{

            width:100px;

            height:100px;

            background:green;

            /* 转换成块*/

            /*display:block;*/

            /*转换成行内块*/

            display:inline-block;

        }

        div{

            width:100px;

            height:100px;

            background:pink;

            /* 这一种不用，你既然把它转换成行内元素了，你直接用行内即可*/

            display:inline;

        }

        p{

            width:100px;

            height:100px;

            background:gold;

            /*让一个元素消失不见了*/

            display:none;

        }

    </style>

</head>

<body>

    <span>我是span</span>

    <i>我是i</i>

    <div>我是div</div>

     <em>我是em</em>

     <p>我是段落p</p>

</body>

</html>

面试题：让一个元素消失的几种方法：

  • display:none

  • visibility:hidden;

  • opacity:0;

三者之间的区别：display：none；彻底没有了，后面的两种，只是看不见了，但是位置还在。
